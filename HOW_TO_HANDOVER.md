## 💡 팁

### 효과적인 전달을 위한 팁

1. **단계적 전달**
   - 먼저 QUICK_HANDOVER.md로 개요 파악
   - 그 다음 HANDOVER_TO_CLAUDE.md로 상세 정보 확인
   - 필요시 개별 파일 참조

2. **구체적인 요청**
   - "도와주세요" 대신 "X를 Y 방법으로 해주세요"
   - 예상 결과물 명시
   - 우선순위 표시

3. **컨텍스트 유지**
   - 대화 중 중요한 정보는 반복해서 언급
   - 파일 경로는 정확하게 명시
   - 명령어는 전체 경로 포함

4. **검증 요청**
   - 작업 후 결과 확인 요청
   - 로그 확인 요청
   - 상태 리포트 요청

## 📞 문제 발생 시

새로운 Claude가 다음과 같은 문제를 겪으면:

### "전략을 이해할 수 없습니다"
→ SimpleAdaptiveStrategy.py 파일의 주석 참조

### "API 오류가 발생합니다"
→ test_bitget_connection.py 실행 결과 공유
